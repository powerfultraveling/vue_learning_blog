經過三天的學習，逐漸掌握了 vue 的基本用法，差不多是時間統整一下腦中的資訊，從摘要中檢測自己的理解程度，一方面也整理自己這三天學習的技術。

## Vue 的特點

### 不是框架容易入門而是讓前端容易入門
與 React 比較，vue 有很多的功能都非常的直覺，舉個例子: 想要設定 data 可以直接用 `this.name = "hallo"` 這種最直觀的方法，不需要像 React 那樣用 setter 來完成。條件式、data-binding、eventlistener 可以直接用 `v-if="show === true"`  這樣長的像是 attribute 的方式完成，感覺很像 HTML 多加上一些功能而已。

React 雖然相較起來比較複雜，但如果是熟悉 JavaScript 的開發者其實使用起來會比較習慣，條件式、資料的寫法其實就是一般的 JavaScript 只是使用大括弧來區別開 HTML template。

對我來說也是這個方法比較熟悉，也不需要去多記憶像是 v-if v-show 的 syntax。所以說，與其說 vue 這個框架比較好入門，倒不如說適合剛入門前端時使用，因為跟 HTML 長得很像，功能也很直覺；但如果對 JS 已經有一定的熟悉度的話，卻不一定是這樣。
